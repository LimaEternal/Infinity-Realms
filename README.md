# DaMS: Infinity Realms — текстовая RPG с AI-мастером

## Общая информация

Текстовая RPG-игра, где роль мастера подземелий исполняет искусственный интеллект. Игрок вводит действия вручную или выбирает из подсказок, а AI генерирует описание мира, управляет инвентарём, эффектами и создаёт уникальные изображения для каждой сцены.

Серверная часть написана на [Flask](https://flask.palletsprojects.com/), обращение к LLM через [OpenRouter API](https://openrouter.ai/) (клиент [openai-python](https://github.com/openai/openai-python)), генерация изображений через [Pollinations AI](https://pollinations.ai/).


## Ключевые аспекты приложения

### Игровой процесс:
- Выбор одного из трёх сеттингов: фэнтези, научная фантастика, постапокалипсис (пользователь также может ввести и свой сеттинг по желанию)
- Свободный ввод действий игрока или выбор из предложенных вариантов
- AI описывает последствия действий, окружение и реакцию мира
- Динамическое обновление инвентаря и эффектов (баффы, дебаффы, статусы)

### Генерация изображений:
- Уникальная иллюстрация для каждой сцены
- Обработка текстового описания от AI для создания промпта
- Соотношение 16:9, модель FLUX

### Интерфейс:
- Дизайн в виде ретро-компьютерного терминала
- Трёхколоночное расположение окон: инвентарь (слева), иллюстрация сцены + чат (центр), эффекты (справа)
- В чате видны только 2 последних ответа мастера — остальные доступны прокруткой
- Адаптивная вёрстка - интерфейс корректно отображается как на узких, так и на широких экранах

### Архитектура проекта:
| Файл | Назначение |
|---|---|
| `config.py` | Flask-приложение, загрузка промптов, API-ключи, `game_state` |
| `openrouter.py` | `clean_json_response()`, вызов OpenRouter API |
| `image_api.py` | `generate_image()` — Pollinations API |
| `app.py` | Flask-маршруты (`/`, `/api/action`, `/api/start`, `/api/image`, `/api/setting`) |
| `static/js/main.js` | Фронтенд-логика: отправка действий, обновление интерфейсы |
| `static/css/style.css` | Стили терминального UI |
| `prompts/` | Системный, начальный и сеттинг-промпты для AI |


## Запуск приложения

Создание виртуального окружения:
```bash
python -m venv venv
```

Активация виртуального окружения:
```bash
./venv/Scripts/activate
```

Установка необходимых модулей:
```bash
pip install flask openai python-dotenv requests
```

Для запуска приложения создайте файл `.env` в корне проекта:
```
SECRET_KEY=your_secret_key_here
OPENROUTER_API_KEY=sk-or-your-api-key
AI_MODEL=qwen/qwen3-coder:free
```

> Получить бесплатный API-ключ для OpenRouter можно на [openrouter.ai](https://openrouter.ai/).

Запуск приложения:
```bash
python app.py
```

Откройте [http://localhost:5000](http://localhost:5000) в браузере и напишите «начать игру».


## Используемая модель

По умолчанию используется `qwen/qwen3-coder:free` — бесплатная модель с хорошим качеством генерации. Вы можете заменить её на любую доступную на OpenRouter, изменив `AI_MODEL` в `.env`.
